# 権限一覧

ALIS OAuth では権限によって利用できる Web API を制限しております。

# 利用できる権限

# read 権限

読み取り専用の権限。一部例外を除き読み取り系の処理(HTTP メソッドでは GET で表現される)のみに制限される。

# write 権限

読み取り/書き込み用の権限。read に加えて、リソースの作成(POST)、更新(PUT)、削除(DELETE)が可能。
