# 権限一覧

ALIS OAuthでは権限によって利用できるWeb APIを制限しております。

# 利用できる権限

# read権限

読み取り専用の権限。一部例外を除き読み取り系の処理(HTTPメソッドではGETで表現される)のみに制限される。

# write権限

読み取り/書き込み用の権限。readに加えて、リソースの作成(POST)、更新(PUT)、削除(DELETE)が可能。
